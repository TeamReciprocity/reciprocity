# reciprocity

A place for sharing recipes, made in Django as a project
for a Code Fellows 401 Python course.

## Cooks
+ Hannah Krager
+ Michael Stokely
+ Michael Sullivan
+ A.J. Wohlfert

Kitchen Prep
------------

- mkdir reciprocity
- cd reciprocity
- virtualenv --no-site-packages reciprocityvenv
- cd reciprocityvenv
- source bin/activate
- pip install django
- git clone https://github.com/TeamReciprocity/reciprocity.git
- get started!
